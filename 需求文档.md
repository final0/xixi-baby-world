# 🌸 汐汐的小窝 - 项目需求文档

> 一个专属于汐汐的温馨成长记录平台 💕

---

## 📌 项目概述

**项目名称**：汐汐的小窝（xixi-baby-world）  
**项目定位**：专属宝宝成长记录平台，集照片、视频、日记、AI助手于一体  
**目标用户**：汐汐的家人（爸爸、妈妈、爷爷奶奶等）

---

## 🧩 功能需求详情

### 1. 注册 & 首次登录 - 角色选择

- 用户注册账号后，**第一次登录**需要选择个人角色
- 角色选项：爸爸 / 妈妈 / 爷爷 / 奶奶 / 外公 / 外婆 / 其他亲属
- 选择角色后，系统根据角色为用户生成**专属小短句**（如"爸爸最爱汐汐啦～"）
- 角色信息保存至用户档案，后续可在个人信息中修改

---

### 2. 个人信息管理

- 登录后可进入**个人信息页**
- 支持功能：
  - 上传 / 更换头像
  - 修改昵称、角色、联系方式等基本信息
  - 查看账号创建时间

---

### 3. 首页展示（温馨欢迎页）

- 登录后进入**专属首页**，内容聚焦汐汐的成长信息
- 展示内容包括：
  - 欢迎语（如：「欢迎来到汐汐的小窝 🌸」）
  - 汐汐的生日 🎂
  - 出生时的体重、身高
  - 精选照片展示
  - 今日/本周专属小短句（滚动形式展示）
  - 温馨背景风格，色调柔和

---

### 4. 内容板块分区展示

- 界面需进行**板块化设计**，不将所有内容堆砌在一起
- 建议分区：
  - 📷 **照片墙**：仅展示图片内容，网格或瀑布流布局
  - 🎬 **视频集**：仅展示视频内容，卡片形式
  - 📝 **成长日记**：文字记录板块
  - 🌟 **专属小短句**：滚动展示区域
- 界面风格精致、温馨，使用柔和配色

---

### 5. 登录 & 注册界面专属化

- 登录页 & 注册页使用**汐汐专属主题**
- 可使用「汐汐」或「xixi」作为专属词融入界面文案
- 示例文案：
  - 登录页：「欢迎回来，快来看看汐汐今天怎么样啦～」
  - 注册页：「加入汐汐的小家庭，一起记录美好时光 💕」
- 背景可使用温馨插画或柔和渐变色

---

### 6. 登录验证 - 图形验证码

- 登录时需完成**图形验证码校验**
- 验证码内容为**加减乘除算术题**（如：`3 + 5 = ?`）
- 防止暴力破解，提升安全性

---

### 7. 注册验证 - 邮箱验证码

- 注册时使用**邮箱验证码**进行身份验证
- 邮件发送方式：**QQ邮箱 SMTP**
- 发送方邮箱：固定配置（系统邮箱）
- 接收方邮箱：用户注册时填写的邮箱
- **邮箱 SMTP 配置**统一在**管理员后台**进行管理，包括：
  - SMTP 服务器地址
  - 端口号
  - 发件邮箱账号
  - 授权码（密码）
  - 是否启用 SSL

---

### 8. AI 聊天助手 - 汐汐小助手 🤖

- 引入 **豆包 API**（字节跳动）作为 AI 聊天能力支持
- **助手名称**：汐汐小助手
- **性格设定（提示词工程）**：
  - 口气俏皮可爱 🌸
  - 总是有礼貌
  - 关心用户，温暖贴心
  - 围绕宝宝成长、育儿话题展开
- **入口形式**：页面右下角**浮动小图标**（可爱风格）
- **首次打开**时助手自动说：「我是汐汐小助手，你要干啥？😄」
- 支持多轮对话，记录当前会话历史

#### 提示词示例（System Prompt）：
```
你是汐汐小助手，是一个专属于汐汐家庭的 AI 助理。
你的性格俏皮可爱、有礼貌、善解人意，非常关心家人的感受。
你喜欢用轻松愉快的语气说话，偶尔加入可爱的表情符号。
你的主要职责是帮助记录宝宝成长、回答育儿问题、给家人带来温暖和快乐。
请始终保持积极、温馨的态度。
```

## 🗂️ 技术建议

| 模块 | 技术建议 |
|------|----------|
| 前端框架 | Vue 3 / React |
| 后端框架 | Spring Boot / Node.js |
| 数据库 | MySQL |
| 邮件服务 | JavaMail + QQ SMTP |
| AI 接口 | Deepseek |
| 图片存储 | Minio / 本地存储 |
| 验证码 | 自定义算术图形验证码 |
